Computer Geek Input Form

Recently I applied for a job at a company called Computer Geek.

After talking with the owner, he agreed to hire me if I could complete a test for him.

For this test, he wanted me to use PHP to write information from an HTML form into a MySQL database and email the results to the respondant. Since I had no prior PHP knowledge, I had to learn all of this from scratch.

I was able to learn and complete the task in under two weeks. The form uses JavaScript to verify information and PHP to run SQL commands as well as sends the email using the PHP mail class.

I already passed the test, but you can still check it out.
