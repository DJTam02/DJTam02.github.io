CeraVela Website

In grade 12, I joined Junior Achievement (JA), a program that allows students to come together to start a small business from scratch.

Like every business, there are different departments and with each department, comes a department head. In both of my JA companies, I was elected as the Head of the IT Department.

While many companies buy a website builder and hosting, I decided to use my skills to save the company some money and code the entire thing from scratch using HTML, CSS, and JavaScript and host it with GitHub pages.

This year, my JA company decided to call ourselves CeraVela, and sell flower shaped candles.

The program is just about done but you can still visit the site.
