Graphing Algorithms Visualizer

Recently I wanted to expand my knowledge of algorithms.

I figured that learning graph traversal algorithms would be the best place to start.

I learned about graphs and how different graph traversal algorithms like Depth First Search, Breadth First Search, and Dijkstra's Algorithm work.

Originally, I wrote my own code to see how close I could get it to the actual code. Even though my first try wasn't exactly the same, I learned a lot about how to make my code more efficient.

But I know it's not enough to just tell you that I know these algorithms. So, I decided to make a web application that allows you to make your own graph, and watch BFS, DFS, and Dijkstra's Algorithm in action.

You can find the demonstration here.
