Welcome to my Graphing Algorithms Visualizer!


I built this web app to display and apply my knowledge of basic graph traversal algorithms.


I also built this to help people understand how these algorithms work through an aesthetic visual demonstration.


This app allows you to create your own simple graph to see Depth First Search, Breadth First Search, or Dijkstra's Algorithm in action!


Please read through this short guide to see how to use this visualizer!

Depth First Search


Depth First Search(DFS) traverses through a graph using a stack.


DFS focuses on going down a single path first. It adds a connected, unvisited node to the end of the stack and pops the node off the end of the stack once all connected nodes have been visited.


To run DFS, click the red button labeled "Depth First Search" on the left of the screen. The stack can be viewed in real-time at the bottom of the screen.

Breadth First Search


Breadth First Search(BFS) traverses through a graph using a queue.


BFS focuses on going across the graph and adding all nodes at a certain number of nodes away from the starting node to the queue.


Once all nodes connected to the first node in the queue have been visited, it is removed from the queue.


To run BFS, click the blue button labeled "Breadth First Search" on the left of the screen.


The queue can be viewed in real-time at the bottom of the screen.

Dijkstra's Algorithm


Dijkstra's Algorithm finds the shortest path to each node by visiting each node and "relaxing" nodes that are "connected".

"Relaxing" is the process of updating the distance of connected nodes by adding the weight of each edge to the current node.

Once a node has been visited, the shortest distance has been found and all connected nodes can then be "relaxed".

You can define the distance/weight of an edge by clicking on an edge and entering a value into the field that shows up.

Dijkstra's Algorithm only works for edges with a weight greater than or equal to 0 so negative weights are not allowed.

To run Dijkstra's Algorithm, click the green button labeled "Dijkstra's Algorithm" on the left of the screen. Each distance can be viewed in real time at the bottom of the screen.

Nodes


Nodes of the graph are represented by circles.


To create a node, drag the circle at the top left of the page to the middle of the screen.


While dragging the circle, you will see that there is a white rectangle with an area of diagonal red stripes surrounding it.


The white rectangle is where you can place your nodes to create your graph. Placing nodes outside of the rectangle will delete them.

Edges

Edges are the lines that connect two nodes.

To create a edge, click with the middle mouse button on one of the nodes, then do the same with another.

(Note: To middle click on a windows laptop, find out how to edit your settings here. If you are on a MacBook, you can install third party software to enable middle clicking here. Middle clicking has many uses so downloading third party software might not be a bad idea.)

To delete edges, middle click on both of the nodes that are connected by the edge you want to delete.

Middle clicking twice in a row on a single node will cancel the request to make an edge with that node (i.e. If you middle click on the wrong node, you can undo your action by middle clicking on the same node).
